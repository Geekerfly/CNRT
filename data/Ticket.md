## 基础票价/ Basic Price

格式化描述意为每增加1元可继续乘坐的公里数或区间数（“[]”内的数字表示区间数，2站=1区间、3站=2区间，依此类推）：
- 0, 0, 6, 6, 10, 10, 20~：起步价3元6公里，每增加1元可继续乘坐6、6、10、10、20公里，且32公里以上每20公里增加1元
- 0, [4], [5], [6], ∞：起步价2元4区间，每增加1元可继续乘坐5、6区间，且15区间以上封顶5元
- 0, 4, 4, 4, 6, 6, 8~：起步价2元4公里，每增加1元可继续乘坐4、4、6、6、8公里，且24公里以上每8公里增加1元
- 0, 5, 5, 5, 7, 7, 9~：起步价2元5公里，每增加1元可继续乘坐5、5、7、7、9公里，且29公里以上每9公里增加1元
- 0, 4, 5, 7, 9, +2~：起步价2元4公里，每增加1元可继续乘坐5、7、9公里，后续每增加1元可乘坐的公里数比上一区段递增2公里
